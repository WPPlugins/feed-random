== Description ==

Adds a feed displaying post randomly.
`http://example.com/feed/random` or `http://example.com/?feed=random`

== Installation ==

1. Upload the entire `feed-random` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
